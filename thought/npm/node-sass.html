# yarn下载vue依赖慢的问题

一些项目在用yarn下载项目的时候很卡或者经常卡在node-sass一直在闪，好久才下载下来，很影响心情，如果我们把镜像配置到国内，你会发现速度会快很多。

下面就是怎么设置：

如果你已经设置好镜像，请忽略第一步和第二步，直接跳到第三步

  1. 首先设置npm或者yarn镜像为淘宝镜像 详情： 淘宝 NPM 镜像
npm config set registry https://registry.npm.taobao.org
OR
yarn config set registry https://registry.npm.taobao.org
1
2
3
  1. 如果觉得这样一个一个设置有点麻烦，我们可以下载nrm 管理我们的镜像
npm install -g nrm
OR
yarn add global nrm
1
2
3

查看是否安装成功:nrm --version

nrm --version

1.0.2
1
2
3

我们可以用 nrm ls 查看我们的源列表

$ nrm ls

* npm -----  https://registry.npmjs.org/
  yarn ----- https://registry.yarnpkg.com
  cnpm ----  http://r.cnpmjs.org/
  taobao --  https://registry.npm.taobao.org/
  nj ------  https://registry.nodejitsu.com/
  skimdb -- https://skimdb.npmjs.com/registry

使用 nrm use 修改镜像

nrm use taobao
1
  1. 在项目的根目录添加.npmrc 或者 .yarnrc 把镜像源设置淘宝镜像，node-sass设置淘宝的路径
registry "https://registry.npm.taobao.org"
sass_binary_site "https://npm.taobao.org/mirrors/node-sass/"
1
2

好了，现在你在重新安装依赖，你会发现下载快了好多。
